## 🚀 Что нового в версии v1.8.9

### 📂 Оптимизация форматов сохранения
* **Универсальный диалог:** По умолчанию предлагается формат `.txt`, но добавлена поддержка профессиональных расширений `.py` и `.pyw`.
* **Чистый интерфейс:** Список типов файлов оптимизирован — удалены лишние 80+ форматов, оставлены только самые востребованные.

### 🌍 Глобальная локализация (360°)
* **Триязычный интерфейс:** Мгновенное переключение между **Русским (RU)**, **English (EN)** и **Українською (UA)** без перезапуска.
* **Триязычный код:** Исходный код полностью задокументирован на трех языках для удобства международного сообщества.

### 📊 Анализ надежности и UX (Smart Analysis)
* **Трехуровневая цветовая индикация:** Обновлена логика индикатора времени взлома (нижняя шкала):
    * 🔴 **4 символа:** Красный (Критически слабо).
    * 🟠 **10 символов:** Оранжевый (Средне).
    * 🟢 **14+ символов:** Зеленый (Безопасно).
* **Time-to-Crack Estimation:** Расчет примерного времени взлома пароля методом брутфорса в реальном времени.
* **Pixel Perfect Icons:** Реализована полная поддержка кастомных иконок щита во всех окнах (Автор, Версия, Уведомления), работающая корректно даже в `.exe`.
* **Adaptive UI:** Полная поддержка системной темной и светлой темы оформления Windows.

## 🛡️ Безопасность и "Zero-Footprint"

* **Изоляция памяти:** Пароли не сохраняются в логи системы и не оставляют следов на диске.
* **Secure Wipe:** Автоматическая очистка буфера обмена через 60 секунд после копирования для предотвращения утечек.
* **Безопасное завершение:** Принудительная очистка буфера обмена Windows при закрытии приложения.
