## 🚀 Что нового в версии v1.8.3

### ⭐ Рейтинг и признание
* **Золотой стандарт**: В интерфейс добавлен фиксированный рейтинг доверия **★★★★★** (5 золотых звезд).

### 🌍 Глобальная локализация
* **Триязычный интерфейс**: Мгновенное переключение между **Русским**, **Английским** и **Украинским** языками прямо во время работы.
* **Динамическое меню**: Исправлена логика обновления верхней панели - теперь названия разделов («Файл», «Опции», «О программе») переводятся корректно.
* **Локализованный код**: Внутренняя документация и комментарии в коде выполнены на трех языках для удобства международного аудита.

### 🛡️ Безопасность и "Zero-Footprint"
* **Изоляция памяти**: Пароли не сохраняются в логи или временные файлы. Программа не оставляет «хвостов» в системе.
* **Безопасное завершение**: При выходе из программы (через меню или крестик) буфер обмена Windows принудительно очищается.
* **Secure Wipe**: Автоматическое удаление пароля из буфера обмена и обнуление внутренних переменных через 60 секунд.
* **Защита от инъекций**: Поле ввода длины пароля проходит жесткую валидацию (`isdigit check`).

### ⚙️ Продвинутый функционал
* **Исключение неоднозначных символов**: Опция для удаления знаков `.,:;\'~"/()[]{}|`, которые могут вызвать проблемы в коде, терминалах или при чтении.
* **Исключение похожих знаков**: Опция убирает путаницу между символами-двойниками (`i, l, 1, L, o, 0`).
* **Скрытие вывода**: Маскировка пароля «звездочками» для защиты от подсматривания (shoulder surfing).

### 📊 Анализ надежности
* **Графический Canvas**: Индикатор сложности с 6-уровневой шкалой оценки в реальном времени.
* **Цветовая схема**: Логика порогов надежности (4, 10, 14 символов) с цветовой индикацией (Красный/Оранжевый/Зеленый).
