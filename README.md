> **Примечание:** Программа поддерживает мгновенное переключение между RU, EN и UA языками через меню «Настройки» / «Settings».

🚀 Что нового в версии v1.8.3
🌍 Локализация и доступность
Мультиязычность: Мгновенное переключение между Русским, Английским и Украинским языками.

Полный перевод: Все меню (включая динамическое «About»), сообщения и окна полностью локализованы.

📊 Индикатор сложности
Графический Progress Bar: Мгновенная оценка надежности пароля.

Цветовая индикация:

🔴 Красный: Уязвимый (до 10 символов).

🟠 Оранжевый: Средний (10-13 символов).

🟢 Зеленый: Надежный (14+ символов).

🛡️ Безопасность
Скрытие символов: Маскировка пароля для защиты от подсматривания.

Автоочистка буфера: Автоматическое удаление пароля из памяти через 60 секунд после копирования.

⚙️ Функционал
Исключение похожих знаков: Опция убирает символы, которые легко перепутать (i, l, 1, L, o, 0).

Экспорт: Сохранение сгенерированных данных через меню «Файл».

DPI Awareness: Четкий интерфейс на мониторах с любым разрешением.

🛠 Установка и запуск
Убедитесь, что у вас установлен Python 3.8+.

Скачайте репозиторий.

Запустите основной файл:
python Secure_Pass_Pro.pyw
Secure_Pass_Pro.exe
