🔐 Secure Pass Pro v1.8.3 Stable

🌍 Глобальная локализация
Триязычный интерфейс: Мгновенное переключение между Русским, Английским и Украинским языками прямо во время работы.

Динамическое меню: Исправлена логика обновления верхней панели — теперь названия разделов («Файл», «Опции», «О программе») переводятся корректно.

Локализованный код: Внутренняя документация и комментарии в коде выполнены на трех языках для удобства аудита.

🛡️ Безопасность и "Zero-Footprint"
Изоляция памяти: Пароли не сохраняются в логи или временные файлы.

Безопасное завершение: При выходе из программы буфер обмена принудительно очищается для защиты конфиденциальных данных.

Secure Wipe: Автоматическое удаление пароля из буфера обмена и обнуление внутренних переменных через 60 секунд.

Защита от инъекций: Поле ввода длины пароля проходит жесткую валидацию (защита от ввода букв и скриптов).

⚙️ Продвинутый функционал
Исключение неоднозначных символов: Новая опция для удаления знаков .,:;\'~"/()[]{}|, которые могут вызвать проблемы в коде или при чтении.

Исключение похожих знаков: Опция убирает путаницу между символами i, l, 1, L, o, 0.

Скрытие вывода: Маскировка пароля «звездочками» для защиты от подсматривания (shoulder surfing).

📊 Анализ надежности
Графический Canvas: Индикатор сложности с 6-уровневой шкалой оценки.

Цветовая схема: Логика порогов надежности (4, 10, 14 символов) с цветовой индикацией (Красный/Оранжевый/Зеленый).

🛠 Установка и запуск / Installation
📥 Для пользователей (Windows)
Перейдите в раздел Релизы.

Скачайте файл 📥 Secure_Pass_Pro.exe.

Программа работает как Portable (установка не требуется).

💻 Для разработчиков
Убедитесь, что у вас установлен Python 3.x.

Склонируйте репозиторий:
git clone https://github.com/Maximka1993271/Password-Generator-Python.git

Запустите основной файл:
python Secure_Pass_Pro.pyw
