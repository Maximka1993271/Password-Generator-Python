🚀 Что нового в версии v1.8.9
📂 Максимальное расширение базы форматов
Универсальный диалог сохранения: Добавлена поддержка более 80 профессиональных форматов файлов.

Широкий спектр расширений: В список включены Python, C++, SQL, Rust, Swift, JSON, YAML, а также специализированные форматы вроде MMIXAL, Forth и Assembly.

Интеллектуальный выбор: Программа позволяет сохранять пароли в файлы, оптимизированные под вашу среду разработки.

🌍 Глобальная локализация (360°)
Триязычный интерфейс: Мгновенное переключение между Русским, English и Українська языками без перезапуска.

Динамическое обновление: Все модальные окна, уведомления и элементы меню («Файл», «Опции», «О программе») переводятся мгновенно.

Международный код: Техническая документация и комментарии внутри исходного кода выполнены на трех языках.

🛡️ Безопасность и "Zero-Footprint"
Изоляция памяти: Пароли не сохраняются в логи или временные файлы; программа не оставляет следов в системе.

Secure Wipe: Автоматическая очистка буфера обмена Windows и обнуление внутренних переменных через 60 секунд после копирования.

Безопасное завершение: При выходе из программы буфер обмена принудительно очищается.

Валидация ввода: Реализована жесткая проверка поля длины пароля (isdigit check).

📊 Анализ надежности и UX
Графический Canvas: Индикатор сложности с 6-уровневой шкалой оценки надежности в реальном времени.

Adaptive UI: Полная поддержка системной темной и светлой темы оформления Windows.

Золотой рейтинг: В футер программы добавлен фиксированный рейтинг доверия ★★★★★.

🛠 Установка и запуск / Installation
📥 Для пользователей (Windows)
Перейдите в раздел **[Релизы](https://github.com/Maximka1993271/Password-Generator-Python/releases)**.

Скачайте файл **[📥 Secure_Pass_Pro.exe](https://github.com/Maximka1993271/Password-Generator-Python/releases/download/SecurePassProv1.8.9/Secure_Pass_Pro.exe)**.

Программа работает как Portable (установка не требуется).

💻 Для разработчиков
Убедитесь, что у вас установлен Python 3.14.4.

Запустите основной файл:

python Secure_Pass_Pro.pyw
