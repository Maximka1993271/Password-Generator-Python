Adaptive QR UI: Окно с QR-кодом полностью поддерживает светлую, тёмную и системную темы, автоматически адаптируясь под настройки основного окна.

📂 Оптимизация форматов сохранения
Универсальный диалог: По умолчанию предлагается формат .txt, но сохранена поддержка профессиональных расширений .py и .pyw.

Чистый интерфейс: Список типов файлов оптимизирован — удалены лишние 80+ форматов для быстрого выбора.

🌍 Глобальная локализация (360°)
Триязычный интерфейс: Мгновенное переключение между Русским (RU), English (EN) и Українською (UA) без перезапуска.

Триязычный код: Весь исходный код снабжен комментариями на трех языках (RU/EN/UA).

📊 Анализ надежности и UX (Smart Analysis)
Трехуровневая цветовая индикация: Обновлена логика визуального анализа (нижняя шкала времени взлома):

🔴 4 символа: Красный (Критически слабо).

🟠 10 символов: Оранжевый (Средне).

🟢 14+ символов: Зеленый (Безопасно).

Pixel Perfect Icons: Полная поддержка кастомных иконок щита во всех дочерних окнах, работающая даже внутри скомпилированного .exe.

Adaptive UI: Бесшовное переключение между темами оформления Windows.

🛡️ Безопасность и "Zero-Footprint"
Изоляция памяти: Пароли не кэшируются и не оставляют временных файлов.

Secure Wipe: Автоматическая очистка буфера обмена через 60 секунд.

Безопасное завершение: Принудительная очистка буфера Windows при закрытии приложения.
